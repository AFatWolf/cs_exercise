# Turtle graphic.  
Basic implementation:  
```
import turtle
kame = turtle.Turtle()
kame.shape('turtle)
```
Move:  
```
kame.forward(150)
kame.backward(150)
kame.goto(150,200)
```  
Go home:  
```
kame.home()
```  
Move without trademark (pen up).  
```
kame.penup()
kame.isdown()
kame.pendown()
```  
Change the direction:  
```
kame.left(angle)
kame.right(angle)
```
